# DAVIS_mini_project
Product rating analysis a simmple mini project for data analysis

1. Introduction

In today’s e-commerce and digital marketplaces, product ratings and reviews play a crucial role in customer purchasing decisions. Analyzing product ratings helps businesses understand customer satisfaction, product quality, and areas for improvement.
This mini project focuses on analyzing product ratings data using data analysis techniques to extract meaningful insights such as average ratings, rating distribution, and top-performing products.


2. Problem Statement

To analyze product rating data and:
Understand customer satisfaction levels
Identify top-rated and low-rated products
Study rating distribution patterns
Provide actionable insights for business decisions


3. Objectives

To load and preprocess product rating data
To analyze rating trends across products
To identify best and worst-rated products
To visualize rating distribution
To summarize findings and insights


4. Dataset Description

The dataset contains the following attributes:

Column Name	Description
Product_ID	Unique ID of the product
Product_Name	Name of the product
Category	Product category
Rating	Customer rating (1 to 5)
Review_Count	Number of reviews


Sample Dataset:
